# form-generator

This package vue3 element-plus form generator.

## How to dev
```
pnpm install
pnpm dev
```

## Install
```
pnpm install vfg
```

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Run Unit Tests with [Vitest](https://vitest.dev/)

```sh
npm run test:unit
```
